## Getting Started

### Prerequisites

- Node.js (v16 or higher recommended)
- npm or yarn

### Installation

1. Install dependencies:

```bash
npm install
```

### Running the Application

Start the development server:

```bash
npm run dev
```

### Configuration

You can configure the backend API URL by creating a `.env` file in the root directory:

```env
VITE_API_URL=http://localhost:8080/api
VITE_APP_TITLE=AI Task Manager
```
